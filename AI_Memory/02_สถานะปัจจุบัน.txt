✦ Routes ที่ทำงานอยู่

  GET  /                  → public/index.html (VP Kanniporn home)
  GET  /login             → public/login.html (FWD orange gradient)
  GET  /profile           → public/profile.html (ต้อง auth) — โปรไฟล์ของตัวเอง
  GET  /agent/:id         → public/profile.html (ดูคนอื่นได้ ไม่ต้อง auth)
  GET  /search            → public/search.html (ค้นหาตัวแทนทั้งหมด)
  GET  /packages          → public/packages.html (placeholder)
  GET  /health            → health check
  POST /api/auth/register → สมัคร + อัปโหลด avatar
  POST /api/auth/login    → login (email + password)
  POST /api/auth/logout
  GET  /api/auth/me       → คืน user object เต็ม (ถ้า login)
  PUT  /api/auth/profile  → แก้โปรไฟล์ + avatar + cover_images[] (max 6)
  GET  /api/users         → list ตัวแทนทุกคน (สำหรับ search)
  GET  /api/users/:id     → public profile
